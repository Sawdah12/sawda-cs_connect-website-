Upcoming Events

CS Connect Pakistan Hackathon

This is an open virtual hackathon for all students across Pakistan! The best project will be hosted on csconnect.pk and the team will be officially recognized!

How to Participate:

  • Fork the GitHub repository
  • Create a new branch using your GitHub username (for teams, use any member's username)
  • Develop your project
  • Open a Pull Request
  • Wait for judges' feedback

Rules:

  • One submission per team (solo participation allowed)
  • Generative AI tools are allowed
  • You may use open-source templates (with proper license respect)
  • Website must include: Home, About Us, Contact Us, Blog sections

Deadline: April 16, 2025
